## Weather API

This repository contains a simple Weather API built with .NET Core 8.0.

### Quick Start

```bash
cd WeatherApi
dotnet run
```

Then open your browser to `http://localhost:5279/swagger` to see the interactive API documentation.

### Features

- RESTful weather forecast endpoint
- 5-day forecast with temperature data (Celsius and Fahrenheit)
- Swagger/OpenAPI documentation
- Built with ASP.NET Core 8.0

For detailed documentation, see the [WeatherApi README](WeatherApi/README.md).
